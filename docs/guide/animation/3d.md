# 3d动画

## 卡片旋转木马
通过设置 景深属性 <code>perspective</code>、3d渲染<code>preseve-3d</code>、z轴投影成图<code>translateZ()</code>、旋转变形<code>rotateY()</code>去实现一个简单的木马旋转特效。
> 大概原理成像是 通过perspective 设置视角远近，translateZ 设置目标与成像屏幕的距离移动，最终投影成像为屏幕显示的大小形状。
